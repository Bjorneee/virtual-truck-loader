# Unity Development Setup

## Step 1
Install unity hub at https://unity.com/download

## Step 2
Create Account/Login to Unity Hub and Install Unity 6.2 (6000.2.10f1)

## Step 3
Clone project repository into a directory on your local machine

### Run:
```bash
git clone https://github.com/Bjorneee/virtual-truck-loader.git
```

## Step 4
Add `unity` folder to Unity Hub project directory
- Go to `Projects` tab
- Click the `Add` dropdown in the top right
- Click `Add project from disk`
- Select `unity` folder from project repo

Open the project and Unity will autogenerate any files that are ignored in the repo

## Important
Please remember to push all changes to a separate branch. DO NOT PUSH TO MAIN.
